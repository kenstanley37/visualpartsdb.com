The array elements are as follow: ptg: The Excel function ptg code. args: The number of arguments that the function takes:

toReversePolish()

toReversePolish(array  $tree = array()) : string

Builds a string containing the tree in reverse polish notation (What you would use in a HP calculator stack).

The following tree:.

+ / \ 2 3

produces: "23+"

The following tree:

+ / \ 3 * / \ 6 A1

produces: "36A1*+"

In fact all operands, functions, references, etc... are written as ptg's

Parameters

array $tree

the optional tree to convert

Returns

string —

The tree in reverse polish notation
